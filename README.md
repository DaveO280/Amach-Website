## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Development Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn/ui
